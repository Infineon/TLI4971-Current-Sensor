SICI.cpp - Part of the library for Arduino to control the tli4971-D050T4 current sensor.

tli4971 is a high-precision current sensor based on InfineonÂ´s proven Hall technology. The coreless concept significantly reduces footprint compared with existing solutions. tli4971 is an easy-to-use, fully digital solution that does not require external calibration or additional parts such as A/D converters, 0 pAmps or reference voltage.

Have a look at the application note/reference manual for more information.
